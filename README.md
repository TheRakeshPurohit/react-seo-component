# react-seo-component

## Props

| Prop            | Use                 |
| --------------- | ------------------- |
| title           | Site title          |
| description     | Page description    |
| pathname        | Page URL            |
| article         | article             |
| image           | image               |
| siteLanguage    | defaults to 'en'    |
| siteLocale      | defaults to 'en_gb' |
| twitterUsername | can be null         |
